# Quote API

A simple API for getting and adding random quotes to a server

## Tech Stack

    - Node.js
    - Express
    - JavaScript
    - HTML & CSS

## Usage:

    - install Node
    - clone repository
    - open folder in console (cmd, bash, shell)
    - type 'npm install'
    - type 'node server.js'
    - open http://localhost:4001/ in your browser
